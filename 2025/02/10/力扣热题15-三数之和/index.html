    15. 三数之和

    问题描述

    给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

    **注意：**答案中不可以包含重复的三元组。

    示例 1：

    输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。

    示例 2：

    输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。

    示例 3：

    输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。

    提示：

    • 3 <= nums.length <= 3000

    • -105 <= nums[i] <= 105

    为了找出整数数组中所有和为 0 且不重复的三元组，可以使用排序加双指针的方法。以下是具体的实现思路和代码：

    思路分析

    1. 排序：首先对数组进行排序，排序的目的是方便后续使用双指针法，并且可以很容易地跳过重复元素。
    2. 遍历数组：固定一个元素 nums[i]，然后在其右侧使用双指针 left 和 right 来寻找满足 nums[i] + nums[left] + nums[right] == 0 的另外两个元素。
    3. 双指针移动：
      • 如果三个数的和小于 0，说明需要增大和，将 left 指针右移。
      • 如果三个数的和大于 0，说明需要减小和，将 right 指针左移。
      • 如果三个数的和等于 0，将这三个数添加到结果列表中，并同时移动 left 和 right 指针，跳过重复元素。
    4. 跳过重复元素：在遍历过程中，需要跳过重复的元素，避免结果中出现重复的三元组。

    代码实现

    def threeSum(nums):
    # 对数组进行排序
    nums.sort()
    result = []
    n = len(nums)
    # 遍历数组
    for i in range(n):
        # 如果当前元素大于 0，由于数组已经排序，后面的元素也都大于 0，不可能找到和为 0 的三元组
        if nums[i] > 0:
            break
        # 跳过重复元素
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        # 初始化双指针
        left, right = i + 1, n - 1
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total < 0:
                # 和小于 0，左指针右移
                left += 1
            elif total > 0:
                # 和大于 0，右指针左移
                right -= 1
            else:
                # 和等于 0，添加到结果列表
                result.append([nums[i], nums[left], nums[right]])
                # 跳过重复元素
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                # 移动指针
                left += 1
                right -= 1
    return result

# 测试示例
nums1 = [-1, 0, 1, 2, -1, -4]
print(threeSum(nums1))  # 输出: [[-1, -1, 2], [-1, 0, 1]]

nums2 = [0, 1, 1]
print(threeSum(nums2))  # 输出: []

nums3 = [0, 0, 0]
print(threeSum(nums3))  # 输出: [[0, 0, 0]]

    复杂度分析

    • 时间复杂度：排序的时间复杂度为 $O(n log n)$，双指针遍历的时间复杂度为 $O(n^2)$，因此总的时间复杂度为 $O(n^2)$。
    • 空间复杂度：主要是排序所需的额外空间，为 $O(log n)$。
