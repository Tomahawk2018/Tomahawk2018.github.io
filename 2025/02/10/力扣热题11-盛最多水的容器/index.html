    11. 盛最多水的容器

    问题描述：

    给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

    找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

    返回容器可以储存的最大水量。

    **说明：**你不能倾斜容器。

    示例 1：

    输入：[1,8,6,2,5,4,8,3,7]
输出：49 
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

    示例 2：

    输入：height = [1,1]
输出：1

    提示：

    • n == height.length
    • 2 <= n <= 105
    • 0 <= height[i] <= 104

    示例代码及注释：

    class Solution(object):
    # 定义一个类Solution，该类包含一个方法maxArea，用于解决容器盛水问题
    def maxArea(self, height):
        # 初始化左指针left，指向height列表的起始位置
        left = 0
        # 初始化右指针right，指向height列表的末尾位置
        right = len(height) - 1
        # 初始化变量max_water，用于记录能容纳的最大水量，初始值为0
        max_water = 0
        # 当左指针left小于右指针right时，执行循环
        while left < right:
            # 计算当前由left和right所指的两条垂线与x轴构成的容器的盛水量
            # 容器的高度由两条垂线中较短的那条决定，宽度为right - left
            current_water = min(height[left], height[right]) * (right - left)
            # 更新max_water的值，使其始终记录当前找到的最大盛水量
            max_water = max(max_water, current_water)
            # 打印当前循环的相关数据
            print(f"当前循环: left = {left}, right = {right}, height[left] = {height[left]}, height[right] = {height[right]}, current_water = {current_water}, max_water = {max_water}")
            # 如果height[left]小于height[right]，将左指针left向右移动一位
            # 因为此时较矮的是左边的垂线，移动左指针可能会找到更大的盛水量
            if height[left] < height[right]:
                left += 1
            # 如果height[left]大于等于height[right]，将右指针right向左移动一位
            # 因为此时较矮的是右边的垂线，移动右指针可能会找到更大的盛水量
            else:
                right -= 1
        # 循环结束后，返回找到的最大盛水量
        return max_water


# 创建Solution类的实例s
s = Solution()
# 定义一个列表height，用于模拟容器的垂线高度
height = [1, 1]
#height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
# 使用%格式化输出输入的数据
print('输入: %s' % height)
# 使用f - string格式化输出输入的数据
print(f'输入： {height}')
# 使用str.format()方法格式化输出输入的数据
print('输入：{}'.format(height))
# 调用s的maxArea方法，计算并输出能容纳的最大水量
print(s.maxArea(height))

    这段代码的整体功能是使用双指针法解决“盛最多水的容器”问题，通过不断调整左右指针的位置，计算不同情况下容器的盛水量，从而找到最大盛水量。同时，代码中还展示了三种不同的Python字符串格式化方法来输出输入的 height 列表数据。

    这里咱们的循环为什么要这样写呢，这样写的目的是什么呢

    在这段计算容器最大盛水量的代码中，while left < right 循环的设计思路如下：

    算法目的

    该算法要解决的是“盛最多水的容器”问题，即给定一个整数数组 height，数组中每个元素代表一条垂线的高度，找出其中两条垂线，使得它们与 x 轴共同构成的容器能够容纳最多的水。

    双指针法思路

    采用双指针法，设置左指针 left 和右指针 right 分别指向数组的起始和末尾位置。容器的盛水量取决于两条垂线中较短的那条（决定了水的高度）以及两条垂线之间的距离（决定了水的宽度），即 min(height[left], height[right]) * (right - left)。

    循环的作用

    通过循环不断调整左右指针的位置，来遍历所有可能的垂线组合情况，从而找到最大的盛水量。在每次循环中：

    • 计算当前指针位置所构成容器的盛水量 current_water。
    • 将其与已记录的最大盛水量 max_water 比较，更新 max_water。
    • 根据两条垂线的高度情况来移动指针：如果 height[left] < height[right]，说明左边的垂线较矮，此时将左指针 left 向右移动一位（因为当前较矮的左边垂线限制了盛水量，向右移动左指针有可能找到更高的垂线，从而得到更大的盛水量）；否则，将右指针 right 向左移动一位（同理，右边的垂线较矮，向左移动右指针可能找到更高的垂线以增大盛水量）。

    循环终止条件

    当左指针 left 与右指针 right 相遇时，即 left >= right，说明已经遍历完所有可能的垂线组合情况，循环终止，此时得到的 max_water 就是能够容纳的最大水量。

    这种循环设计能够以 $O(n)$ 的时间复杂度高效地解决问题，避免了使用嵌套循环带来的 $O(n^2)$ 的高时间复杂度，大大提高了算法效率。
