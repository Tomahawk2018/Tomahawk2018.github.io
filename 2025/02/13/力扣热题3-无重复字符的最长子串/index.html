    力扣热题3-无重复字符的最长子串

    练习 练习 力扣 学习
    发布时间 : 2025-02-13 14:15
    阅读 :
    1. 代码说明

    3. 无重复字符的最长子串

    提示

    给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串的长度。

    示例 1:

    输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

    示例 2:

    输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

    示例 3:

    输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

    提示：

    • 0 <= s.length <= 5 * 104

    • s 由英文字母、数字、符号和空格组成

    
class Solution(object):
    # 定义一个方法lengthOfLongestSubstring，用于计算字符串中最长无重复字符子串的长度
    def lengthOfLongestSubstring(self, s: str) -> int:
        # 创建一个集合char_set，用于存储当前窗口内已经出现过的字符
        char_set = set()
        # 定义左指针left，初始值为0，表示窗口的左边界
        left = 0
        # 定义变量max_length，用于记录最长无重复字符子串的长度，初始值为0
        max_length = 0
        # 使用for循环遍历字符串s，right为当前遍历到的字符的索引
        for right in range(len(s)):
            # 当当前字符s[right]已经在集合char_set中时，说明出现了重复字符
            while s[right] in char_set:
                # 移除窗口左边界的字符，即s[left]
                char_set.remove(s[left])
                # 左指针向右移动一位，缩小窗口
                left += 1
            # 将当前字符s[right]添加到集合char_set中
            char_set.add(s[right])
            # 更新最长无重复字符子串的长度，取当前窗口长度（right - left + 1）和max_length中的较大值
            max_length = max(max_length, right - left + 1)
        # 返回最长无重复字符子串的长度
        return max_length


# 创建Solution类的实例s
s = Solution()
# 定义字符串s1
s1 = "abcabcbb"
# 定义字符串s2
s2 = "bbbbb"
# 定义字符串s3
s3 = "pwwkew"
# 调用lengthOfLongestSubstring方法计算s1中最长无重复字符子串的长度并打印
print(s.lengthOfLongestSubstring(s1))
# 调用lengthOfLongestSubstring方法计算s2中最长无重复字符子串的长度并打印
print(s.lengthOfLongestSubstring(s2))
# 调用lengthOfLongestSubstring方法计算s3中最长无重复字符子串的长度并打印
print(s.lengthOfLongestSubstring(s3))

    代码说明

    1. 类定义：
      • class Solution(object)：定义一个名为Solution的类，继承自object类（在Python 3中，所有类默认继承自object）。
    2. 方法定义：
      • def lengthOfLongestSubstring(self, s: str) -> int:：在Solution类中定义一个方法lengthOfLongestSubstring，该方法接受一个字符串s作为参数，并返回一个整数。
    3. 变量初始化：
      • char_set = set()：创建一个空集合char_set，用于存储当前窗口内出现的字符，以检查是否有重复字符。
      • left = 0：初始化左指针left为0，代表滑动窗口的左边界。
      • max_length = 0：初始化变量max_length为0，用于记录最长无重复字符子串的长度。
    4. 遍历字符串：
      • for right in range(len(s)):：使用for循环遍历字符串s，right为当前遍历到的字符的索引。
    5. 处理重复字符：
      • while s[right] in char_set:：当当前字符s[right]已经在集合char_set中时，说明出现了重复字符，进入while循环。
      • char_set.remove(s[left])：从集合char_set中移除窗口左边界的字符s[left]。
      • left += 1：左指针向右移动一位，缩小窗口，直到窗口内不再有重复字符。
    6. 更新集合和长度：
      • char_set.add(s[right])：将当前字符s[right]添加到集合char_set中。
      • max_length = max(max_length, right - left + 1)：更新最长无重复字符子串的长度，取当前窗口长度（right - left + 1）和max_length中的较大值。
    7. 返回结果：
      • return max_length：返回最长无重复字符子串的长度。
    8. 测试部分：
      • 创建Solution类的实例s。
      • 定义三个测试字符串s1、s2和s3。
      • 分别调用lengthOfLongestSubstring方法计算并打印每个字符串中最长无重复字符子串的长度。
